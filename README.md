## 🚀 Features

- **Live Jolly** — Restaurant brand with full menu & cart
- **Live Mart** — Grocery e-commerce with product filters
- **Live Real Estate** — Property enquiry system
- **Live Logistics** — Fleet & delivery services
- **Live Tech** — Technology division showcase
- **Shopping Cart** — Add to cart, quantity control, checkout modal
- **Contact Page** — Full enquiry form
- **Responsive Sidebar** — Mobile-friendly navigation
- **Customer Reviews** — Live review submission system
